pub fn gossip_round(&mut self, messages: Vec<(u64, Bytes)>) -> usize

Runs a round of gossiping.

messages is a list of (sender, message) pairs. All messages will be sent simultaneously. The round will run until all peers received all messages, or until SimulatorConfig::gossip_round_timeout is elapsed.

Returns the number of undelivered messages.
